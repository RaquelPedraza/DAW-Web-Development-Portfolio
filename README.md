# DAW - Web Development Portfolio

Bienvenido a mi portfolio de desarrollo web. Este repositorio documenta mi evolución técnica y los proyectos prácticos realizados durante el Grado Superior en **Desarrollo de Aplicaciones Web (DAW)**.

Actualmente, el repositorio refleja mi enfoque en el ecosistema **Frontend Moderno (Vue.js, SASS, UI/UX)** mientras consolido las bases de lógica de programación y Backend.

## Tech Stack Actual
* **Frontend Core:** HTML5, CSS3, JavaScript (ES6+).
* **Estilos & UI:** SASS/SCSS, Bootstrap, Diseño Responsive, Mobile First.
* **Frameworks:** Vue.js.
* **Backend:** PHP (Fundamentos), Gestión de formularios.
* **Sistemas:** Linux (Comandos), Apache, Configuración de servidores.

---

## Estructura del Portfolio

El contenido está organizado por áreas de competencia técnica:

### 01-Interfaces-UI-UX
*Mi especialidad actual.* Colección de prácticas avanzadas de maquetación y diseño web.
* **Tecnologías:** SASS, Bootstrap, CSS Grid/Flexbox, Media Queries.
* **Destacado:** Integración de elementos dinámicos y adaptación a múltiples dispositivos.

### 02-Frontend-Frameworks-Vue
Prácticas utilizando arquitectura basada en componentes.
* **Tecnologías:** Vue.js, Directivas, Manejo de estado.
* Desarrollo de interfaces reactivas y modernas.

### 03-JavaScript-Core
Fundamentos sólidos de programación y lógica.
* **Enfoque:** Programación Orientada a Objetos (POO), Clases, Herencia y manipulación del DOM.
* Ejercicios de sintaxis ES6+ y algoritmos.

### 04-Backend-PHP
Introducción al desarrollo en entorno servidor.
* **Tecnologías:** PHP nativo.
* Procesamiento de formularios y lógica básica de servidor.

### 05-Docs-Despliegue
Documentación técnica sobre administración de sistemas.
* Guías de configuración de servidores Apache en Linux.
* Uso de clientes FTP y comandos de terminal.
